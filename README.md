# 🍽️ Zomato Restaurant Analysis

Exploratory Data Analysis of 51,717 Zomato Bangalore restaurants using Python, Pandas, NumPy and Matplotlib. Also includes an interactive Power BI Dashboard.

## 📊 View Notebook
👉 [Click here to view the full notebook](https://nbviewer.org/github/DaweAnki/zomato-restaurant-analysis/blob/main/Zomato_analysis.ipynb)

## 📊 Power BI Dashboard Preview
![Dashboard](zomato%20Dashboard.png)

## 📌 Key Insights
- Analyzed 51,717 restaurants across Bangalore
- Explored ratings, cuisines, costs, and online ordering trends
- Identified top locations and popular restaurant types
- Built interactive Power BI dashboard with KPI cards and filters

## 🛠️ Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Microsoft Power BI
- Jupyter Notebook

## 📁 Dataset
Dataset from [Kaggle - Zomato Bangalore Restaurants](https://www.kaggle.com/datasets/himanshupoddar/zomato-bangalore-restaurants)
